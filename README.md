🛠 Available Scripts
In the project directory, you can run:

npm start ->
Runs the app in development mode at http://localhost:3000. The page reloads automatically on save.

npm test ->
Launches the interactive test runner.

npm run build -> 
Bundles the app into static files for production in the build/ folder. It optimizes the build for the best performance.

npm run eject -> 
Note: This is a one-way operation. Removes the single build dependency and moves all configuration files (webpack, Babel, etc.) directly into your project for full control.
